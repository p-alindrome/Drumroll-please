# Drumroll-please 🥁 🥁 🥁
Making decisions… dramatically...🥁🥁🥁✨✨✨

A **fun, interactive website** to randomly choose a person, task, or decision in a playful way!  

Perfect for:  
- Picking a winner from a group  
- Deciding who does a task  
- Choosing between friends or ideas  
- Or just being silly 😏  

---

## 🚀 Live Demo

Check out the live version here:

👉 https://p-alindrome.github.io/Drumroll-please/

This demo shows the full **Drumroll, Please** experience — add items, clear with fun animations, and pick a dramatic winner with confetti!

___

## 🌈 Features

- **Add / Remove / Clear All items** 📝  
- **Grid layout** for your list 📋  
- **Winner reveal** dramatic on stage 🏆  
- **Colorful confetti** explodes around the winner 🎊  
- **Clear All** with **blurry poof + dust particles** ✨  
- Fully **interactive and persistent** with localStorage 💾  

---

## ⚡ How to Use

1. **Add items**  
   - Type a name, task, or idea in the input box  
   - Click **Add**  

2. **Clear all items**  
   - Click **Clear All** next to the Add button  
   - Watch the **poof + dust animation**  

3. **Pick a winner**  
   - Click **Start the Drumroll**  
   - Drumroll plays 🥁  
   - Winner appears on stage with **colorful confetti**  
   - Optional: Remove winner from the list automatically  

---

## 🎨 Customization

- Change **theme colors** in CSS (currently purple/pink)  
- Adjust **confetti particles** count/duration in `confetti()`  
- Grid layout **responsive** on desktop and mobile  
- Add **sound effects** or extra animations for more fun
